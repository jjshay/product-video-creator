## [1.0.0] - 2025-01-11

### Added

- Ken Burns effect video generation from static product images
- Smooth pan and zoom animations with natural easing
- Elegant crossfade transitions between shots
- 45-second video structure with opening, detail shots, quotes, and closing segments
- Text overlay support for art quotes and authenticity badges
- Library of 15+ inspirational art quotes
- Color-matched backgrounds based on artwork palette analysis
- HD output at 1920x1080 resolution and 30fps
- Batch processing for generating videos across entire inventory
- Google Drive automatic upload integration
- Support for 8 product photo inputs per video (full image plus detail shots)
- Sample artwork crops and video configuration for demonstration
- Core video generation functions in dedicated module
- Demo mode that works without external dependencies
- Visual showcase script for quick demonstration
- Comprehensive test suite
- CI workflow for automated testing
